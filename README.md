# GreenFlowerSite

--NOTE! - This isn't a real site. I created it to train HTML and CSS skills--

This site is a startup for GreenFlower Company which is able to deliver a lot of kind of flowers to your home.
If you buy a subscription you will get a random flower every week.

There are 3 kinds of subscription:\
  -> Classic - you are gettting a random flower every week\
  -> Advanced - you are able to choose between 2 flowers, which were chosen for you randomly and get a small free gift\
  -> FlowerLover - you are able to choose between 3 flowers, which were chosen for you randomly, get medium free gift and discount for next month of subscription

Site shows what kind of flowers you can recieve and short description of categories.
There is also a gallery which shows few example flowers.
